# 📞 Sentiric Telephony Action Service

Agent'tan gelen emirleri alır, media-service ile köprü kurar ve AI döngüsünü işletir.

## 🚀 Hızlı Başlangıç
```bash
cargo run --release
```

## 🏛️ Mimari ve Mantık
* **Geliştirici Kuralları:** Gizli [.context.md](.context.md) dosyasını okuyun (AI Ajanları için zorunludur).
* **İş Mantığı ve Algoritmalar:** [LOGIC.md](LOGIC.md) dosyasını inceleyin.
* **Anayasal Konum:** [sentiric-spec/spec/services/telephony-action.spec.yaml](https://github.com/sentiric/sentiric-spec)
